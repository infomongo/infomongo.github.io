About Me

I‘m John Phillips, a UX Designer & Web Developer with 15 years experience designing and building for the web. On the twitter, I'm @infomongo.

I enjoy creating prototypes. Prototypes are a quick way to explore an idea and get a feel for it. I use a couple of frameworks to speed the process up: Jekyll a ruby framework for creating static sites and Angular JS a JavaScript framework for creating applications.

Angular Apps

This ROI Calculator is a simple application created using Angular. I did the UX Design and much of the engineering, adapted from an existing project. It’s a simple marketing tool rather than a prototype, but it shows capabilities of Angular. Total dev time for this project was about 2 weeks.

Store Locators

At Placeable we created a platform that allowed us to quickly create store locators for our clients. I did the UX design of the templates used in the system. These were then “skinned” with the sites branding. I was also part of the engineering team that built the system.

Some example sites:

  • Smashburger – We created this locator for the brand in 2014.
  • Big Lots – A locator for the discount retailer. Try searching in Denver, CO or click the link “Browse by State” and follow the SEO links.
  • UMB Bank – A branch and ATM locator for his bank. Try searching Hartford, CT.

Each of these sites has a mobile version, try visiting them on your phone.

On each of these sites the branding and graphic design was done by others. I did the UX design of the base templates, including the flow between pages and information hierarchy.

Bigger Brands

In addition to the standard templates, clients could choose to fully customize the sites. Bigger brands often went this route.

  • Nationwide Insurance - I worked with Nationwide’s excellent UX team to design this Agent Locator. The basic flow is similar to the sites above, but it also supports searching by name and provides a wealth of details about each location. The mobile version of this site is slightly simpler, as our research revealed different use cases for mobile users.
  • Western Union – This one was designed by their in-house team. The mobile variation uses the standard designs I developed.
